## 実装コマンド

dump
DB内容をファイルシステムに書き出す。

check
DB内容とファイルシステムの一致確認を行う(存在確認のみ)

export
DB内容を単一ファイルに書き出す。
形式はCSVで。ストリームを使って処理する(多分メモリに載り切らない。)
バイナリはbase64エンコードで処理

import:db
exportで処理した形式のデータをDBに展開する。
一行づつ読み込む形でトランザクションは使わない(DB負荷がヤバい)

import:file
exportで処理した形式のデータをファイルストレージにのみ
一行づつ読み込む形でトランザクションは使わない(DB負荷がヤバい)
